Variables

If you are using the Sass version you can personalize Super GiGi simply changing the value of its variables before importing main.sass.
For the size variables, you would like to have the rem/em-calc() functions available. To achieve that remember to import the _functions.scss partial before.

$rem-base Link

Default:16px

Type:unit

$row-width Link

Default:rem-calc(1920)

Type:unit

$column-gutter Link

Default:rem-calc(30)

Type:unit

$grid-columns Link

Default:12

Type:number

$use-classes Link

Default:false

Type:boolean

$use-placeholders Link

Default:false

Type:boolean

Default:row

Type:string

$column-name Link

Default:column

Type:string

$grid-start Link

$breakpoints Link

Type:map
